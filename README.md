Medieval Platformer is a 2d platformer made in unity as part of a group final for a software engineering class.

this game is a collaborative project made in an effort to test skills in team cooperation and working on a shared project. 
The game itself plays similar to older games such as mario and mega man with elements of combat similar to castlevania.

All artwork is either from a free sprite pack called "Brackeys' Platformer Bundle" or made by the team in pixel art studio to match the art style.
the sprite pack can be found here: https://brackeysgames.itch.io/brackeys-platformer-bundle
